## How to Capture Screenshots

### For Web (Chrome)

1. **Run the app:**
   ```bash
   flutter run -d chrome
   ```

2. **Open Chrome DevTools:**
   - Press `F12` or right-click â†’ Inspect

3. **Enable Device Toolbar:**
   - Click the device icon (or press `Ctrl+Shift+M`)

4. **Select Device:**
   - Choose device from dropdown (e.g., iPhone 12, iPad, Desktop)

5. **Capture Screenshot:**
   - Click the three dots menu in DevTools
   - Select "Capture screenshot"
   - Or use browser extension like "Full Page Screen Capture"

6. **Save with correct name:**
   - Save to this `screenshots/` folder
   - Use exact names listed above

### Recommended Resolutions

- **Mobile**: 375x667 (iPhone SE) or 414x896 (iPhone 11)
- **Tablet**: 768x1024 (iPad)
- **Desktop**: 1920x1080
